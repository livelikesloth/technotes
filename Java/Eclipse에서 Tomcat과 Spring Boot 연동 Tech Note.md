Eclipse에서 Tomcat과 Spring Boot 연동 Tech Note
====
**Eclipse EE**를 사용하게 되면, 내부에 Tomcat을 가지고 있으나, 본 문서는 **Tomcat**을 **직접** 다운받아 **Spring boot**와 연동하는 과정을 정리하였다.

사전 준비될 환경
-----
1. Eclipse Mars
2. JDK 1.8
3. Tomcat 9
4. Spring boot

Spring boot 설치
-----
* [SPRING INITIALIZR](http://start.spring.io/)를 통해서 프로젝트 스케폴딩
